Documentation

This repository documentation can be found at the `COSIMA 3D Animation <http://cosima.github.io/3D_animations/>`_ website.

To learn more about BlenderNC and Blender, follow the links below:

- Blender at https://www.blender.org
- Blender tutorials at https://cloud.blender.org/training/

Contributing & Animation requests

**All contributions and ideas are welcome.** You can request and provide examples by using the `Github repository <https://github.com/COSIMA/3D_animations/>`_.

Aditionally, you can report bugs, suggest animations, and provide examples at the `Github repository issues <https://github.com/COSIMA/3D_animations/issues>`_.
