3D animations¶

This repository contains examples of 3D animations generated using blender and BlenderNC.

Each folder included within this repository has the following structure:

  • Example Name

    • example_blender_file.blend

    • example_animation.png

    • example_animation.rst

All the animations can be found in YouTube in a link contained in each example_animation.rst file. For easier access to the animations and their documentations, refer to the the COSIMA 3D Animation website. All the animations contained in this repository were generated using VDI and data located in gdata.

Aditionally, you can report bugs, suggest animations, and provide examples at the Github repository issues.
